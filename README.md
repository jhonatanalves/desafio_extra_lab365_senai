# Project: EDA de Ride-Booking (NCR)

Projeto profissional de Análise Exploratória de Dados utilizando o dataset `ncr_ride_bookings.csv`.

## Estrutura do Projeto

- `data/`: Contém o dataset original.
- `notebooks/`: Notebooks Jupyter para análise e visualização.
- `src/`: Scripts Python para funções auxiliares e processamento de dados.
- `requirements.txt`: Dependências do projeto.

## Como Iniciar

1. Crie um ambiente virtual:
   ```bash
   python -m venv venv
   ```
2. Ative o ambiente virtual:
   - Windows: `.\venv\Scripts\activate`
   - Linux/Mac: `source venv/bin/activate`
3. Instale as dependências:
   ```bash
   pip install -r requirements.txt
   ```
4. Inicie o Jupyter Lab ou Notebook:
   ```bash
   jupyter lab
   ```
