# Análise Exploratória de Dados — Uber Ride Bookings 2024

Este notebook apresenta uma Análise Exploratória de Dados (AED) desenvolvida como atividade prática extra do curso **Introdução à Inteligência Artificial 
(IP 20h A)** do programa SCTEC/Carreira Tech do SENAI SC LAB365.

O dataset utilizado é o "Uber Ride Analytics Dashboard", disponibilizado publicamente no Kaggle, composto por 150.000 registros de corridas referentes 
ao ano de 2024. A análise busca compreender padrões de utilização do serviço, variações temporais, comportamentos de cancelamento e métricas de avaliação, 
com vistas a identificar possíveis aplicações em modelos introdutórios de Machine Learning.

## Estrutura do Projeto

- `data/`: Contém o dataset original.
- `notebooks/`: Notebooks Jupyter para análise e visualização.
- `requirements.txt`: Dependências do projeto.

## Como Iniciar

1. Crie um ambiente virtual:
   ```bash
   python -m venv venv
   ```
2. Ative o ambiente virtual:
   - Windows: `.\venv\Scripts\activate`
   - Linux/Mac: `source venv/bin/activate`
3. Instale as dependências:
   ```bash
   pip install -r requirements.txt
   ```
4. Inicie o Jupyter Lab ou Notebook:
   ```bash
   jupyter lab
   ```
